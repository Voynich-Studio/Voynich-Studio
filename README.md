## О себе

Я начинающий Android-разработчик, который развивает собственную экосистему мобильных приложений под брендом **Voynich Studio**.

Моя цель — создавать полезные Android-приложения, которые решают реальные повседневные задачи пользователей, работают стабильно, не перегружены лишними функциями и уважают приватность.

Сейчас я фокусируюсь на приложениях с прикладной логикой, расчётными алгоритмами, локальным хранением данных и понятным пользовательским интерфейсом.

## Основные проекты

### Корзина КБЖУ

Android-приложение для анализа продуктовой корзины, расчёта калорий, белков, жиров и углеводов, а также планирования питания.

**Статус:** подготовка к публикации в Google Play.

**Основные возможности:**

* расчёт КБЖУ продуктовой корзины;
* анализ продуктов;
* пользовательские продукты;
* локальное хранение данных;
* офлайн-работа;
* отсутствие рекламы;
* отсутствие регистрации.

**Технологии:**

`Java` `Android SDK` `Gradle` `SharedPreferences` `Material Design`

---

### Калькулятор выслуги лет

Android-приложение для автоматизации расчёта выслуги лет.

Проект демонстрирует работу с датами, периодами, расчётной логикой и пользовательским интерфейсом для специализированной задачи.

**Статус:** в разработке.

**Технологии:**

`Java` `Android SDK` `Gradle` `Date calculations`
